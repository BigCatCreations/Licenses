# Licenses
This repository was designed to manage licenses for commisions by Amy of BigCatCreations.

## How to sign a license?
1. Amy will send you the specific file to edit
2. Review the license
3. Edit the file, adding your name, and the date of your "digital signature"
4. Submit your edit as a pull-request
5. Amy will then acccept the pull-request, and the contract will be signed.
